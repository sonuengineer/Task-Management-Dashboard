# React Task Management Dashboard

A mini **Task Management Dashboard** built with **React.js** and **Vite**, featuring:

- Dashboard overview with total and pending tasks
- Tasks list with filters, search, sorting, and pagination
- Create Task form with auto-focus and toast notifications
- Profile page with editable bio
- Reusable components and clean layout
- Horizontal task cards for better UX
